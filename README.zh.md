# 📘 单词与汉字学习应用

这是一个简单但强大的应用，使用闪卡帮助你记忆单词和汉字。

---

## 🧠 功能特色

- 🎴 使用闪卡学习单词和汉字  
- 🗂 内置词典编辑器：轻松添加、修改和删除词条  
- ⚙️ 灵活设置，适应不同学习场景  
- 🌍 支持20多种语言界面  
- 💡 可以使用 AI 根据示例生成词典  
- 📥 支持从 JSON 文件导入词典  
- 🈯 虽然主要用于日语学习，但也支持其他语言的词汇学习

---

## 💡 示例词典（用于导入）

（JSON 格式）
```
{
  "title": "Японские слова",
  "cards": [
    {
      "word": "猫",
      "translation": "кошка, кот",
      "romaji": "neko"
    }
  ],
  "sentence_mode": false
}
```
